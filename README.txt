-------------------------------------------------------------
	НАЗНАЧЕНИЕ ПРОЕКТА
-------------------------------------------------------------
Простое консольное приложение для работы с удаленным REST API. 

Требования/функционал:
• Возможность отправить запрос на удаленный API (адрес и описание приведено ниже)
• Метод API возвращает 2 списка - продукты и категории
• Полученный ответ приложение должно вывести в консоль согласно примеру:


Product name	Category name

Laptop		Computers
Bread		Food


Описание API:
• адрес: http://tester.consimple.pro
• метод: GET

-------------------------------------------------------------
	КАК ЗАПУСТИТЬ
-------------------------------------------------------------

Открыть папку с проектом.
Открыть файл "ConsimpleApp.exe", путь к которому: 
ConsimpleApp -> ConsimpleApp -> bin -> Debug -> netcoreapp3.1

-------------------------------------------------------------
	КАК ПОЛЬЗОВАТЬСЯ
-------------------------------------------------------------

После запуска программы откроется консольное окно, в котором будет выведена информация с сайта "http://tester.consimple.pro" о продуктах и категориях. 

Данные на сайте ежесекундно обновляются, поэтому в программе есть позвожность обновить таблицу с продуктами и категориями - для этого необходимо ввести <Y> или <y>, затем ENTER.

Для того, чтобы выйти из программы, необходимо дважды нажать ENTER.
